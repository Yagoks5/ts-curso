# Design Patterns (Behavioral) - TypeScript examples

Este diretório contém exemplos didáticos de padrões comportamentais em TypeScript.

Padrões incluídos:

- Strategy
- Observer
- Command
- Iterator
- Mediator
- Memento
- State
- Chain of Responsibility

## O que são padrões comportamentais

Padrões comportamentais (behavioral) tratam das interações e responsabilidades
entre objetos — ou seja, de como objetos se comunicam e distribuem tarefas.
Eles ajudam a organizar o fluxo de controle e a comunicação, tornando o
sistema mais flexível e desacoplado.

Para que servem:

- Organizar a comunicação entre objetos e entidades do sistema.
- Encapsular algoritmos e variações de comportamento (ex.: Strategy).
- Fornecer mecanismos para notificação/assinatura (Observer), histórico
  de estado (Memento), ou sequenciamento de responsabilidades (Chain).

Cada arquivo neste diretório contém um comentário no topo explicando o
respectivo padrão e quando usá-lo.
