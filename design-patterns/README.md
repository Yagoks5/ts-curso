# Design Patterns (Creational) - TypeScript examples

This folder contains TypeScript examples for common creational design patterns:

- Singleton
- Builder
- Factory Method
- Abstract Factory
- Prototype

How to build and run (requires Node.js and npm/yarn):

1. Install dev dependencies:

   npm install

2. Build:

   npm run build

3. Run:

   npm start

Or run the root file directly with ts-node if you prefer.
